Permite ocultar/revelar un mensaje en una imagen.

Documentación de las funciones

void Ocultar ( unsigned char  buffer[],
const char  ocultar[] 
)

Oculta texto en una imagen y se almacena en un array de char.

Parámetros
bufferVector donde se guardará la imagen con el texto ya ocultado.
ocultarTexto a ocultar en la imagen.
Precondición
buffer debe ser una zona de memoria suficientemente grande como para que se pueda ocultar todo el texto.
void Revelar ( unsigned char  buffer[],
char  revelar[],
const int  MAX 
)

Revela texto oculto de una imagen y se almacena en otro vector.

Parámetros
bufferVector donde se encuentra la imagen junto con el texto oculto.
revelarContiene el texto revelado de la imagen.
MAXSe refiere al máximo de caracteres que puede contener el vector del texto revelado.
Precondición
revelar debe ser una zona de memoria suficientemente grande como para que se pueda almacenar todo el texto.
